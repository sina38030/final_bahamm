🔧 دیباگ API جدید تشخیص رهبر

📋 راهنمای استفاده:

  1. سرور backend را اجرا کنید: cd backend && python start_backend.py
  2. توکن معتبر خود را در فیلد زیر وارد کنید
  3. دکمه‌های مختلف را تست کنید
  4. نتایج را در قسمت پایین مشاهده کنید
💡 از localStorage مرورگر (F12 → Application → Local Storage → auth_token) کپی کنید

📄 نتایج:

🔍 نکات مهم:

  • اگر 404 دریافت کردید، سرور backend اجرا نشده است
  • اگر 401 دریافت کردید، توکن نامعتبر است
  • اگر 200 دریافت کردید، API کار می‌کند!
  • برای تست واقعی، از توکن معتبر استفاده کنید
