STEP 1: UPLOAD FILES TO SERVER (2 minutes)

  ./deploy-to-server.sh YOUR_VPS_IP root


STEP 2: RUN SETUP ON SERVER (5 minutes)
----------------------------------------

  ssh root@YOUR_VPS_IP
  cd /root/staging-setup
  chmod +x setup-staging.sh
  sudo ./setup-staging.sh


STEP 3: ADD DNS RECORDS (10 minutes including propagation)
-----------------------------------------------------------

In your DNS provider (e.g., Cloudflare, Namecheap):
  
  Type: A     Name: staging          Value: YOUR_VPS_IP
  Type: A     Name: staging-api      Value: YOUR_VPS_IP

Wait 5-10 minutes for DNS propagation.


STEP 4: (OPTIONAL) SETUP SSL (3 minutes)
-----------------------------------------

  ssh root@YOUR_VPS_IP
  sudo certbot --nginx -d staging.bahamm.ir -d staging-api.bahamm.ir
