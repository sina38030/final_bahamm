===============================================
BACKEND CORS UPDATE INSTRUCTIONS FOR ngrok
===============================================

After you start ngrok and get your URLs, you need to add them to the CORS configuration.

FILE TO EDIT: C:\Projects\final_bahamm\backend\main.py

FIND THESE TWO SECTIONS (around lines 98 and 134):

Section 1 (line 98):
--------------------
api_app.add_middleware(
    CORSMiddleware,
    allow_origins=[
        "http://localhost:3000",
        "http://127.0.0.1:3000",
        "http://localhost:8000",
        "http://127.0.0.1:8000",
        "https://bahamm.ir",
        "http://bahamm.ir",
        "https://app.bahamm.ir",
    ],

Section 2 (line 134):
--------------------
app.add_middleware(
    CORSMiddleware,
    allow_origins=[
        "http://localhost:3000",
        "http://127.0.0.1:3000",
        "http://localhost:8000",
        "http://127.0.0.1:8000",
        "https://bahamm.ir",
        "http://bahamm.ir",
        "https://app.bahamm.ir",
    ],

ADD YOUR ngrok URLs TO BOTH SECTIONS:
-------------------------------------
Replace YOUR_FRONTEND_URL and YOUR_BACKEND_URL with your actual ngrok URLs.

Example of updated section:
    allow_origins=[
        "http://localhost:3000",
        "http://127.0.0.1:3000",
        "http://localhost:8000",
        "http://127.0.0.1:8000",
        "https://bahamm.ir",
        "http://bahamm.ir",
        "https://app.bahamm.ir",
        # ngrok URLs for local testing (REMEMBER TO UPDATE THESE!)
        "https://abc123.ngrok-free.app",  # Frontend ngrok URL
        "https://xyz789.ngrok-free.app",  # Backend ngrok URL
    ],

IMPORTANT:
- Update BOTH sections (api_app and app)
- Use the EXACT URLs from ngrok (including https://)
- Don't add trailing slashes
- Restart your backend server after making changes

After updating, restart the backend:
cd C:\Projects\final_bahamm\backend
uvicorn main:app --reload --port 8080
